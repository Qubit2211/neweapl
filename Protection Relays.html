Protection Relay

Protection Relay Also known as single phase preventers, they help the user to monitor the functionality of given power parameters and control the same by taking appropriate action through relay output. Both analog and digital versions are available. While analog versions come with few parameters and factory set values, digital versions are more versatile helping the user to select the parameters and limits. These units display functional faults and help user to monitor the instant values while functioning faultlessly
